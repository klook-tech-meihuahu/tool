# 更新日志 (Changelog)

本项目所有的重要更改都将记录在此文件中。

## [v1.1.0] - 2026-03-04

### ✨ 新增 (Added)
* **开发备忘录 (DevReference) 模块**：全新上线开发者速查手册，集成到左侧主导航栏。
* **MySQL 进阶速查表**：
  * 新增 MySQL 核心数据类型（数值、日期/时间、字符串）的详细清单，包含存储大小、有效范围及使用建议。
  * 补充经典建表、修改列、增删索引的标准化 SQL 语句片段。
  * 增加高价值避坑指南，例如 `TIMESTAMP` 的 2038 年溢出问题预警。
* **Redis 进阶指南**：
  * 收录 Redis 核心数据结构（String, Hash, List, Set, ZSet, Bitmap, HyperLogLog, Geo）。
  * 深度解析每种结构的**底层实现**（如 SDS、ZipList、SkipList）、**时间复杂度**及**经典适用场景**。
* **发版窗口微调功能**：在 Git 分支生成器中，为 `release` 分支新增发版日期智能推算与手动微调控制台，支持通过 `+` / `-` 按钮在有效的发版日（周二、周四）之间自由偏移切换。
* **内置更新日志弹窗**：引入 `marked.js`，支持在不离开当前页面的情况下，以弹窗形式优雅地渲染和阅读本地 Markdown 格式的更新日志。

### 优化 (Changed)
* **侧边栏布局升级**：在左侧导航栏底部区域重构了安全提示 UI，并新增了“版本历史”和“GitHub”的快捷入口。
* **发布分支计算逻辑**：优化了目标发版日期的 ISO 周序号推演算法，使其能够完美响应用户的自定义偏移量。

## [v1.0.0] - 初始版本

### ✨ 新增 (Added)
* **JSON 全能工具**：支持 JSON 格式化、压缩、转义及去转义。
* **JSON/XML 对比 (Diff)**：支持双栏高亮差异对比，提供差异位置小地图快速定位。
* **XML 全能工具**：支持 XML 报文的格式化排版与压缩。
* **SQL 语法校验**：基于 AST 的实时 SQL 语法错误检查与精准行/列定位。
* **JSON 转 Java**：支持一键将 JSON 报文转化为标准的 Java 实体类 (Bean) 代码。
* **Git 分支生成器**：支持标准化的 feature/bugfix/hotfix/release 分支命名生成，自动处理全小写与中划线格式。
* **常用开发工具**：集成 Base64 编解码、URL 编解码以及在本地/HTTPS 环境下的 SHA-256 实时哈希计算。
