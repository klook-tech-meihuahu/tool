## 📅 更新日志 (Changelog)

本项目所有的重要更改都将记录在此文件中。

### [v1.2.0] - 2026-03-05

#### ✨ 新增 (Added)
* **文本批量编辑替换工具 (Text Replacer)**：全新上线，提供类 IDE 的高级编辑体验。
  * **多模式查找替换**：支持纯文本及正则表达式（全局）的高级替换，完美识别并处理 `\n`、`\t` 等特殊转义字符。
  * **高频预设指令**：内置“换行转逗号”、“逗号转换行”、“去除所有空白”、“单双引号互转”等开发中常用的文本清洗预设，一键执行。
  * **专业级编辑视图**：纯原生实现带有**实时行号展示**的编辑器，支持行号与文本区的丝滑同步滚动，极大提升长文本阅读体验。
  * **智能搜索与高亮导航**：实时统计匹配数量，提供 `上一个/下一个` 快捷跳转。基于原生 API 驱动光标精准选中，并自动平滑滚动居中定位目标词。
* **正则表达式测试 (Regex Tester)**：新增独立的正则调试面板。
  * 包含常用正则速查（邮箱、手机号、身份证、URL、IP 等）并支持一键填入。
  * 支持 `g`（全局）、`i`（忽略大小写）、`m`（多行）修饰符的可视化动态切换。
  * 智能高亮匹配结果，并自动解析、提取并展示深层匹配的捕获组 (Capture Groups) 详情及索引位置。

### [v1.1.0] - 2026-03-04

#### ✨ 新增 (Added)
* **开发备忘录 (DevReference) 模块**：全新上线开发者速查手册，集成到左侧主导航栏。
* **MySQL 进阶速查表**：

  * 增加高价值避坑指南，例如 `TIMESTAMP` 的 2038 年溢出问题预警。

* **发版窗口微调功能**：在 Git 分支生成器中，为 `release` 分支新增发版日期智能推算与手动微调控制台。
* **内置更新日志弹窗**：支持在不离开当前页面的情况下，以弹窗形式优雅地渲染和阅读 Markdown 格式的文档。

### [v1.0.0] - 初始版本

#### ✨ 新增 (Added)
* **核心工具上线**：JSON/XML 全能工具（格式化/压缩/转义）。
* **代码对比 (Diff)**：支持 JSON 与 XML 的双栏高亮差异对比，提供差异位置小地图快速定位。
* **SQL 语法校验**：基于 AST 的实时 SQL 语法错误检查，支持精准行/列定位。
* **JSON 转 Java**：支持一键将 JSON 报文转化为标准的 Java 实体类 (Bean) 代码。
* **Git 分支生成器**：支持标准化的分支命名生成，自动处理全小写与中划线格式。
* **常用开发工具**：集成 Base64 编解码、URL 编解码以及 SHA-256 实时哈希计算。

## 🌟 项目特色 (Features)

* ⚡ **极速响应**：全站基于纯前端技术栈构建，无后端接口请求，所有计算均在浏览器本地完成，响应时间 < 10ms。
* 🛡️ **隐私安全**：代码 100% 运行在本地环境，不上传任何敏感 JSON 数据、SQL 报文或业务代码，杜绝数据泄露风险。
* 🎨 **开发者友好**：内置 IDE 级别的交互体验，支持行号同步、正则表达式高亮、JSON 结构树等功能。
* 🚀 **离线可用**：支持 PWA 特性，可作为桌面应用安装，在无网络环境下依然保持全功能运作。
